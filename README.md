# Matrix Path Exploration

Matrices are often defined as a map that takes vectors and outputs vectors, usually in different dimensions.

This project explores Square matrices, and the notion of "Paths" that an input v takes to reach the output Av.

**Is there only one path? does a path uncover meaning or useful interpretations? can it become a tool for interpreting systems that rely on matrix multiplication as their core operation, such as Neural Networks?**

This project tries to explore these questions.
